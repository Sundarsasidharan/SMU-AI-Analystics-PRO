🚀 SMU AI Analytics Dashboard

📌 Overview

An AI-powered Data Analytics Dashboard built using Python and CustomTkinter. 
This tool allows users to upload datasets, clean data, perform analysis, and generate intelligent insights automatically.

🔥 Key Features

- 📂 Upload CSV / Excel files
- 🧹Automate Data Cleaning Module
- Automate Processing
- 📊 Exploratory Data Analysis (EDA)
- Auto Visuvalization
- 🤖 AI Insights Generation
- 📈 Interactive Dashboard UI
- Export

🛠️ Tech Stack

- Python
- CustomTkinter
- Pandas
- Matplotlib

▶️ How to Run

python Analytics.py
